### Run it by yourself

```sh
git clone https://github.com/h4cktivist/jewelry-store.git
cd jewelry-store
pip install - r requirements.txt

python app.py
```

### Run it with Docker
```sh
docker pull h4cktivist/jewelry-store
docker run -p 5000:5000 h4cktivist/jewelry-store
```
